for creating username :git config --global user.name "name of user"
for creating usermail :git config --global user.mail "your mailid"
for creating a git repository :git init
for creating a git repository with branch main:git init -b main
for add a file in git repository :git add <FileName>
for commint :git commit -m "message"
for checking log in commandpromt :git log
for checking status :git status
direct commiting command: git commit -a -m "message"
for removing file in git: git rm --cached <FileName>
for tagging :git tag -a vx.y -m "message"
for pushing a file into repository : git push UrlName BranchName [eg: git push origin main]
